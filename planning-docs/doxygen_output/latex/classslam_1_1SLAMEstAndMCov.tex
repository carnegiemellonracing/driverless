\doxysubsection{Detailed Description}
Maintains and updates landmark estimates and marginal covariance matrices using i\+SAM2. 

This class provides a lgihtweight interface around GTSAM\textquotesingle{}s i\+SAM2 for managing SLAM estimates and covariance matrices for landmarks (cones), which are continuously updated during the first lap. It supports partial and full recalculation, cone-\/based proximity updates, and Mahalanobis distance computation.
